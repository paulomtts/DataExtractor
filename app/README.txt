# EXTERNAL DEPENDENCIES #########################

JavaRuntime (for running Tika)
Anaconda (Python library package)


# WHAT DOES THIS PROGRAM DO? ####################

Extracts data from text-like PDF files based on
provided regular expressions. The data extraction
capabilities depend enormously on the quality and
feasibility of the expressions.

Images will not be processed, this tool is meant
for text only.
